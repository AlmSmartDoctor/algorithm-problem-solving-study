##무식하게 풀기(Brute Force)
-------------------------------------

Brute Force는 가능한 경우의 수를 일일이 나열하면서 CPU의 성능을 통해 답을 찾는 방법입니다.

모든 방법을 다 탐색하므로 무식하다는 표현이 들어갔다고 합니다.


### 재귀호출과 완전 탐색
-------------------------------------





#### 예제 : 보글 게임
-------------------------------------













### 최적화 문제
---------------------------------------------
책의 내용을 대충 요약하면 위의 방법에서 중복을 없애는 것이 핵심입니다!!
