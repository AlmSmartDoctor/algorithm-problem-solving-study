# 알고리즘 문제풀이 스터디

## 규칙

> 시니어는 이론 발표 면제입니다. **문제 풀이는 예외 없이 전원 필수입니다.**
>
> 피치 못 할 사정으로 발표날 휴가를 쓴 경우에는 스터디를 한 회차 미룹니다.

- 문제 푼 코드 문제 발표날 당일 출근 전까지 커밋하기
- 발표 준비가 미흡한 경우: 스타벅스 커피
- 문제 풀이가 미흡한 경우: 매머드 커피
